# Introduction

Prompt Engineering is an important technique in Artificial Intelligence that helps improve the interaction between users and AI systems. The quality of responses generated by Large Language Models depends heavily on how prompts are written and structured.

Naïve prompts are broad, vague, or unstructured prompts that provide little context or guidance to the AI model. Basic prompts are clearer, more detailed, and structured prompts that help the AI understand the task more accurately.

This experiment compares the responses generated by ChatGPT for naïve prompts and refined basic prompts across multiple scenarios such as creative writing, factual answering, summarization, and recommendation systems.

The study helps evaluate how prompt clarity influences response quality, accuracy, reasoning depth, and overall usefulness.

---

# Objectives

1. To understand different prompting patterns.
2. To compare naïve prompts and basic prompts.
3. To evaluate AI-generated responses.
4. To analyze response quality, depth, and accuracy.
5. To identify effective prompt engineering strategies.

---

# Theory

## Naïve Prompt

A naïve prompt is simple, broad, or unclear and lacks detailed instructions.

### Example:

“Tell me about AI.”

### Characteristics:

* Short and unstructured
* Minimal context
* Less control over response

---

## Basic Prompt

A basic prompt is refined, structured, and provides clear instructions to the AI model.

### Example:

“Explain Artificial Intelligence including definition, history, applications, advantages, and disadvantages in simple terms.”

### Characteristics:

* Detailed instructions
* Clear context
* Better response quality

---

# Procedure

1. Define different prompt types.
2. Select multiple test scenarios.
3. Create both naïve and refined prompts for each scenario.
4. Input prompts into ChatGPT.
5. Record generated responses.
6. Compare responses based on quality, accuracy, and depth.
7. Analyze how prompt structure affects output quality.
